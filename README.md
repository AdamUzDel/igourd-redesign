# iGourd - Advanced POS Solutions Website

## Project Description

iGourd is a cutting-edge Point of Sale (POS) solutions provider, offering advanced systems for businesses of all sizes. This repository contains the source code for the iGourd corporate website, built using Next.js 15 and featuring multi-language support, responsive design, and a modern user interface.

## Features

- Multi-language support (English and Chinese)
- Responsive design for mobile and desktop
- Dynamic routing with Next.js App Router
- Server-side rendering and static site generation
- Interactive UI components using shadcn/ui
- Customizable theming with Tailwind CSS
- Accessibility-focused development

## Key Pages

- Home page with product showcase and key features
- Products and Services overview
- Training and Support center
- Contact Us page with form submission
- Careers / Join Us page
- Privacy Policy page

## Technologies Used

- Next.js 15.1.4
- React 18
- TypeScript (85.4%)
- JavaScript (8.8%)
- Tailwind CSS
- shadcn/ui components
- Lucide React icons

## Prerequisites

- Node.js 18.x or later
- npm 9.x or later
